Project Overview


NALCO Drive is designed to offer a user-friendly platform for file management. It includes features like file uploading, downloading, sharing, and secure storage.




Technologies Used


  1. HTML5
    • The foundation of our web pages, providing structure and meaningful content.
  2. CSS3
    • For styling and visual appeal, ensuring a responsive and aesthetically pleasing user interface.
  3. JavaScript
    • Enabling interactivity and dynamic content updates, enhancing user experience.
  4. Bootstrap 4
    • A powerful front-end framework that helps create a responsive and mobile-first design.
  5. PHP 7
    • Server-side scripting to handle the backend logic, database interactions, and secure data processing.
  6. XAMPP
    • An easy-to-install Apache distribution containing PHP and MySQL, used for local development and testing.
